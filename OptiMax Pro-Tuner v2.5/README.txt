============================================================
           OPTIMAX PRO-TUNER v2.5 - USER GUIDE
============================================================

OptiMax Pro is an elite-tier Windows performance utility designed 
to minimize system latency and maximize hardware throughput.

------------------------------------------------------------
[!] CRITICAL: RUN AS ADMINISTRATOR
------------------------------------------------------------
Because OptiMax interacts directly with the Windows NT Kernel, 
Power Schemes, and CPU Priority queues, it MUST be launched 
with Administrator privileges to function.

Right-click 'OptiMax_Profiler.exe' -> Run as Administrator.

------------------------------------------------------------
QUICK START STEPS:
------------------------------------------------------------
1. Launch the application.
2. Navigate to 'Control Center'.
3. Click 'Run Core Tweaks' to engage high-resolution timers 
   and network unchoking.
4. Use 'Dynamic AutoBoost' to automatically engage elite 
   settings when your specific game executable launches.
5. Use the 'Integrity Benchmark' to audit your system's 
   exact DPC/Interrupt latency stability.

------------------------------------------------------------
TROUBLESHOOTING:
------------------------------------------------------------
* Windows SmartScreen: If Windows flags the app as 'Unknown', 
  click "More Info" -> "Run Anyway".
* Reverting Changes: Click "Restore Factory Defaults" in the 
  Dashboard to revert all system-level locks and power plans.
